# RegistroDigitale

Progetto di Registro Digitale sviluppato per e con la classe 3Â°B del Liceo Scientifico "G.Galilei" di Lamezia Terme, consiste nella realizzazione di un registro per i docenti che invece di apporre una firma, dimostrano la propria presenza tramite la lettura della loro impronta. Per la realizzazione si utilizzano un Raspberry pi (zero w), Arduino, lettore di impronte(FPS_GT511C3) e uno schermo lcd 16x2.
